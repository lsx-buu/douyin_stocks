# A股策略回测质量门禁

## 先判数据粒度

| 策略主题 | 最低数据粒度 | 推荐数据粒度 |
| --- | --- | --- |
| 均线、K线、支撑压力 | 日线 | 分钟线 |
| 换手率、相对成交量 | 日线 | 分钟线 |
| 量比 | 分钟线 | 分钟线加历史同时间成交 |
| 集合竞价 | 竞价数据 | 竞价委托和成交明细 |
| 委比、订单簿 | L1盘口 | L2订单簿 |
| 内外盘、主动买卖 | 逐笔成交 | 逐笔成交加盘口 |
| 涨停后分时路径 | 分钟线 | 分钟线加封单和逐笔成交 |
| 尾盘强度和隔夜溢价 | 分钟线 | 分钟线加收盘竞价明细 |
| 龙虎榜和异常波动公告 | 公告时间戳 | 公告时间戳加次日竞价数据 |
| 产业涨价和预期差 | 日线加事件时间戳 | 产品价格、库存、公告、财报和分析师预测 |

## 必须检查的偏差

- 生存者偏差：退市股、长期停牌股是否保留。
- 未来函数：量比、换手率、最高最低、支撑压力是否用了未来数据。
- 交易时间：信号生成时点是否早于下单时点。
- T+1：买入后是否错误地当天卖出。
- 涨跌停：涨停买不到、跌停卖不出是否模拟。
- 停牌：停牌期间是否错误成交。
- 交易成本：佣金、印花税、滑点、冲击成本。
- 容量：成交额太小的标的是否无法承载资金。
- 公开信息时点：龙虎榜、异常波动、财报、产业公告是否只在公开后使用。
- 样本选择：只挑知名游资、热门题材或成功人物案例会产生幸存者偏差。

## 因子验证顺序

1. 单变量分组：按分位数看未来收益、胜率、波动。
2. 中性化：控制市值、价格、行业、波动率、流动性。
3. 样本外：按年份滚动验证。
4. 市场状态：牛市、熊市、震荡市分别验证。
5. 交易成本后收益：看净收益，而不是毛收益。
6. 稳定性：参数附近是否平滑，不要只找最优点。
7. 组合相关性：与已有因子是否高度重合。

## 最小可交付结果

每个策略研究至少输出：

- 策略假设。
- 数据字段。
- 信号计算时点。
- 交易规则。
- 样本范围。
- 成本假设。
- 年化收益、最大回撤、换手率、夏普。
- 分年度表现。
- 失效场景。
- 对应的反证问题和反证结果。

## 否决条件

出现以下情况，不进入策略库：

- 只在单一年份有效。
- 只在极小市值或无法成交样本有效。
- 不考虑涨跌停后收益消失。
- 参数稍微变化结果就崩。
- 信号需要未来全天成交量。
- 解释依赖“主力一定会怎样”，但没有可观测代理变量。
- 盘后公开信息被用于当日盘中交易。
- 涨停、跌停、一字板样本没有真实可成交处理。
- 没有进入 [[50_实验验证/策略假设反证队列|策略假设反证队列]]，或无法提出清晰反证问题。
