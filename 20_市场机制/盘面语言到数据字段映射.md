# 盘面语言到数据字段映射

## 目的

把视频里的“盘感词”翻译成可搜索、可提取、可回测的数据字段。

## 字段映射表

| 盘面语言 | 可观测字段 | 搜索方向 |
| --- | --- | --- |
| 分时走势 | 分钟价格、VWAP、均价线、分时收益率 | intraday return, VWAP deviation |
| 成交量放大 | 分钟成交量、成交额、相对成交量 | relative volume, abnormal volume |
| 量比 | 当前成交量相对历史同时间均值 | intraday volume profile, relative volume |
| 换手率 | 成交量 / 流通股本 | turnover ratio, float turnover |
| 委比 | 买盘挂单量与卖盘挂单量差值占比 | order imbalance, queue imbalance |
| 内盘外盘 | 主动卖出量、主动买入量 | trade classification, buyer initiated volume |
| 成交明细大单 | 单笔成交额、成交方向、价格冲击 | large trades, price impact |
| 集合竞价 | 竞价成交价、竞价成交量、未成交委托 | opening auction, call auction imbalance |

## 转译原则

- “主力吸筹”不是字段，可以拆成大单买入、价格冲击小、成交量放大但价格窄幅波动。
- “主力出货”不是字段，可以拆成主动买入多但价格不涨、放量滞涨、盘口卖压增强。
- “骗线”不是字段，可以拆成信号触发后短窗口反向收益、撤单率异常、突破失败。
- “承接强”不是字段，可以拆成下跌时成交量放大但价格回撤受限，或卖单冲击后快速回补。
- “龙头信仰”不是字段，可以拆成板块内排序、带动效应、次日溢价和负反馈阈值。
- “买分歧卖一致”不是字段，可以拆成分歧状态、修复强度、一致拥挤度和后续收益分布。
- “确定性”不是感觉，可以拆成历史同类事件的胜率、盈亏比、最大回撤和可成交性。
- “明天涨停的票”不是预测目标本身，可以拆成前一日尾盘强度、近期涨停事件、开盘强度、板块共振和次日可成交收益。
- “卡异动、黑屋、监管”不是交易规则本身，可以拆成公告事件、交易所风险提示、高位涨幅、连板高度和公告后的收益分布。
- “游资席位来了”不是无条件买点，可以拆成公开交易信息、市场注意力、连续上榜、题材热度和次日竞价溢价。

## 数据粒度要求

- 日线足够：换手率、均线、K线、指数 regime。
- 分钟线更好：量比、分时趋势、VWAP 偏离、开盘行为。
- Tick 或 L2 必须：委比、内外盘、订单簿、撤单、队列位置。
- 公告和公开信息必须有发布时间：龙虎榜、异常波动、风险提示、财报和产业公告只能在公开之后使用。
