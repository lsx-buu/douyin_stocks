## 使用原则

- 先把视频里的主观盘感转成数据字段，再搜索论文、研报、开源代码或数据集。
- 优先寻找可回测、可复现、可解释的策略线索。
- 不把“主力”“骗线”“承接”等词直接当事实；它们只作为可观测代理变量的命名入口。
- 所有策略假设都必须经过样本外验证、交易成本、T+1、涨跌停、停牌和容量约束检查。
